Content Manager : Политика конфиденциальности

# Политика конфиденциальности для Content Manager

Для компании MultiPlayer одним из главных приоритетов является конфиденциальность наших посетителей. В данном документе, посвященном политике конфиденциальности, указаны типы информации, которую собирает и записывает Content Manager, а также порядок ее использования.



Если у вас возникнут дополнительные вопросы или потребуется дополнительная информация о нашей политике конфиденциальности, обращайтесь к нам.



# Лог-файлы

Само приложение Content Manager не создаёт файлы журналов ни на серверах MultiPlayer, ни на вашем устройстве. Но файлы журнала может создавать и собирать Google YouTube Api.



# Какие данные мы собираем

Наше приложение базируется на том, чтобы помогать авторам контента на площадке YouTube анализировать как свой контент, так и контент других авторов.

Далее Content Manager - Приложение компании MultiPlayer, YouTube Api - Официальная API платформа для работы с YouTube.



Для полноценной работы приложения потребуется ваш Google Account для авторизации в приложении Content Manager и/или YouTube Api. Все данные передаются в Live-режиме из Content Manager к YouTube Api и наоборот, при этом сохраняются в оперативной памяти вашего устройства и/или передаются напрямую в выводимый текст без сохранения в оперативную память. Также эти данные не передаются нам.



# Какие возможности требуется для полноценной работы приложения

Для полноценной работы приложения требуются следущие возможности: Получение имени канала (вашего и/или другого автора), Подключение к сайту https://youtube.com/ для установки идентификатора канала по ссылке, получение кол-во просмотров (на ролике и/или на канале и/или на вашем канале), получение кол-во лайков и дизлайков (на видео и/или на комментариях), получение информации о доступности комментариев на ролике, получение информации о комменатриях (Имя написавшего комментария, кол-во лайков, текст комментария, дата публикации, получение самого популярного комментария, получение последнего комментария), получение дата публикации (канала и/или ролика и/или комментария), получение информации о статусе видео (Является ли видео для детей), получение ID видео, получение прошедшего времени с момента публикации (видео и/или комментария и/или канала), получение кол-во видео (на канале и/или на вашем канале), получение кол-во подписчиков (на канале и/или на вашем канале), установка ссылки видео (по айди видео).



Также, чтобы все возможности приложения были доступны, перед использованием приложения убедитесь, что на площадке YouTube, на вашем аккаунте есть созданный канал.



# Политика конфиденциальности третьих лиц

Политика конфиденциальности Content Manager не распространяется на других рекламодателей или веб-сайты. Поэтому для получения более подробной информации мы рекомендуем обращаться к соответствующим политикам конфиденциальности этих сторонних рекламных серверов. Она может включать информацию об их методах работы и инструкции о том, как отказаться от определенных опций.



# Информация для детей

Еще одним нашим приоритетом является обеспечение защиты детей при использовании Интернета. Мы призываем родителей и опекунов наблюдать за детьми, участвовать в их жизни и/или контролировать и направлять их действия в Интернете.



Content Manager сознательно не собирает персональные данные детей младше 14 лет. Если вы считаете, что ваш ребенок предоставил такую информацию в нашем приложении, мы настоятельно рекомендуем вам немедленно связаться с нами, и мы приложим все усилия, чтобы оперативно удалить такую информацию из наших записей.



# Политика конфиденциальности только в Интернете

Настоящая Политика конфиденциальности распространяется только на нашу деятельность в Интернете и действует для посетителей нашего Приложения в отношении информации, которую они предоставляют и/или собирают в Content Manager. Данная политика не распространяется на любую информацию, собранную в автономном режиме или через каналы, отличные от данного приложения.



# Согласие

Используя наше приложение, вы тем самым выражаете согласие с нашей Политикой конфиденциальности и принимаете ее Условия и положения.
